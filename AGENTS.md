# 羽毛球活动助手开发规则

本项目第一版是手机端 H5 单页应用，使用 Vite + React + TypeScript。

## 开发原则

- 简洁优先：用最少、最直接的代码完成当前功能。
- 精准修改：只修改与当前任务直接相关的文件和逻辑。
- 不引入不必要依赖：第一版不使用复杂 UI 框架或状态管理库。
- 不做第一版范围外功能：暂不开发登录、后端、云同步、数据库、微信支付、比分记录、随机组队、人员名单管理、胜率统计。
- 计算公式不能随意修改：费用计算规则必须保持产品需求中的公式。
- localStorage 是第一版唯一数据存储方式。

## 第一版功能边界

- 费用计算
- 女生优惠金额
- 详细计算过程开关
- 保存历史记录
- 查看历史记录
- 历史记录再次计算回填
- 删除历史记录
- 复制收款文案
- 我的设置页
